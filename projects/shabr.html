Shabr Project

The purpose of the SHABR project is to use bootstrap and current knowledge of css/html to make a website. The content of this project will be provided by the teacher and we will need to make a website with the content given. For bootstrap, we need to pick 6 bootstrap pieces to learn and apply it to the website. First I needed to plan out how the website will look like before I actually make it. Once I planned out how the finishing product of the website will look like I started to code. On my way of making this project a challenge I faced was learning on my own. The bootstrap will not be taught but will need to be learned by ourselves. I used most of my time learning bootstrap so I know how to use it and where to apply it to. From finishing this project I learned a lot of new bootstrap pieces and by this I can apply what I learned to other projects in the future.
